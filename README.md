## Shua
一个好看又耐操的代刷系统

#### 特点
- 开源免费
- 基于 [Lumen](https://lumen.laravel.com/) 开发，优越的性能，优雅的代码
- 严格遵守 [PSR](https://psr.phphub.org/) 规范开发，对眼睛、脑子、新手都好
- 前后端分离，便于开发的同时减轻服务器压力
- 细致的权限系统，不知道有啥用
